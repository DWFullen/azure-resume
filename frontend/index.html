    Azure Resume Challenge

    Github • Site • September 2022

    The Azure Resume Challenge by A Cloud Guru was a cloud challenge that I saw while completing the Indiana State Earn and Learn program(SEAL). The project was a perfect opportunity to create a capstone for everything I have learned while completing the program.

  • The frontend is a provided template that has been modified that holds the HTML and CSS.
  • The backend is a serverless API using Azure Functions to do a HTTP trigger.
  • The 'visitor' counter is utilizing Azure CosmosDB to keep track of the count.
  • Azure Blob Storage is storing the frontend. Azure CDN is also being utilized for static content delivery.
  • CI/CD setup via Github actions with triggers being set on the 'frontend' and 'backend' folders.
